// There are five elements in Arknights: Endfield. Physical stands apart — it builds Vulnerable stacks and bypasses the Arts system entirely. The other four apply Arts Inflictions that stack up to 4 times.

// Arts Reactions fire when a second Arts element is applied to an enemy that already has a different Arts Infliction active. The triggering element is what determines the reaction.

CONCEPT — Arts Burst
Same Element

Applying the same element repeatedly builds Infliction stacks up to 4. Each new stack triggers an Arts Burst — an immediate bonus hit of that element's damage. Running a mono-element team maximizes Burst uptime since every attack contributes to the same stack.

CONCEPT — Arts Reaction
Mixed Elements

Applying a second, different element to an enemy that already has active Inflictions consumes those stacks and fires an Arts Reaction. Mixed-element teams trade consistent Burst damage for powerful situational reactions — particularly Electrification or Corrosion.

// Every effective team in Endfield answers the same three questions: what stack type are we building, who spends those stacks for burst damage, and who keeps the engine running.
